# Gemini AI Assistant Guidelines

## Tool Usage Preferences

When retrieving documentation for Go packages or symbols, **always prefer to use the `godoc` tool** once it is configured and available.

Only use `WebFetch` or `GoogleSearch` for Go documentation if:
*   `godoc` is not configured or available.
*   `godoc` does not provide a clear or sufficient answer.

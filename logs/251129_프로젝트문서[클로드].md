# 이카운트 조직도 설정 가이드

## 1. 부서 등록

### 부서 체계 구성
- 기본설정 → 회사정보 → 부서관리
- 부서코드: 자동생성 또는 수동입력
- 부서명: 명확하고 간결하게
- 상위부서: 계층구조 설정

### 권장 부서 구조
```
본사
├── 경영지원부
├── 영업부
├── 개발부
└── 생산부
```

## 2. 직위/직급 설정

### 직급 등록
- 기본설정 → 회사정보 → 직급관리
- 직급코드 및 직급명 입력
- 순서: 높은 직급부터 낮은 직급 순

### 일반적인 직급 체계
1. 대표이사
2. 이사
3. 부장
4. 차장
5. 과장
6. 대리
7. 주임
8. 사원

## 3. 직원 정보 등록

### 필수 입력사항
- 사원번호: 자동생성 또는 수동입력
- 성명 및 주민등록번호
- 입사일자
- 부서 및 직급
- 근무형태 (정규직/계약직)

### 추가 정보
- 연락처 (휴대폰/이메일)
- 주소
- 급여정보
- 계좌정보

## 4. 조직도 구성

### 조직도 설정 순서
1. 부서 체계 먼저 구성
2. 직급 체계 설정
3. 직원별 부서/직급 배치
4. 결재라인 설정

### 결재라인 설정
- 각 부서별 결재권자 지정
- 대결자 설정
- 전결규정 설정

## 5. 권한 설정

### 메뉴별 권한
- 직급별 기본권한 설정
- 개인별 추가권한 부여
- 데이터 조회/입력/수정/삭제 권한

### 보안 설정
- 급여정보 접근권한
- 인사정보 열람권한
- 재무정보 접근권한

## 6. 실무 팁

### 효율적인 관리방법
1. 부서코드는 체계적으로 (예: 10-경영, 20-영업)
2. 직급은 너무 세분화하지 말 것
3. 퇴사자는 삭제보다 퇴사처리
4. 정기적인 조직도 업데이트

### 주의사항
- 개인정보보호법 준수
- 권한설정 신중히
- 백업 주기적으로
- 변경이력 관리

## 7. 자주 묻는 질문

**Q: 겸직 처리는?**
A: 주부서 설정 후 겸직부서 추가등록

**Q: 임시직 관리는?**
A: 별도 고용형태로 구분하여 관리

**Q: 조직개편 시?**
A: 날짜별 이력관리 기능 활용

**Q: 파견직원은?**
A: 소속회사 구분하여 별도 관리

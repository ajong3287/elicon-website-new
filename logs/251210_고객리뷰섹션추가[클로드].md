## 리뷰 인센티브 프로그램 상세

### 프로그램 구조

**목적**: 리뷰 작성률 증대 → 사회적 증명 강화 → 구매 전환율 향상

**인센티브 설계**:
```
텍스트 리뷰 (5,000원)
├── 조건: 100자 이상 작성
├── 적립: 즉시 지급
└── 사용: 다음 구매 시

포토/동영상 리뷰 (10,000원)
├── 조건: 착용 사진 또는 영상 포함
├── 추가: 텍스트 100자 이상 병행
└── 효과: 신뢰도 2배 향상 (시각적 증거)
```

**기대 효과**:
- 리뷰 작성률: 현재 10% → 목표 30% (3배 증가)
- 포토 리뷰 비율: 현재 5% → 목표 20% (4배 증가)
- 월간 리뷰 수: 현재 20개 → 목표 50개 (2.5배 증가)

**ROI 계산**:
```
비용: 월 500,000원 (50개 리뷰 × 평균 10,000원)
효과: 구매 전환율 +5% → 월 매출 +2,000,000원 (ROI 4.0)
장기: 신뢰도 누적 → 브랜드 자산 형성 (ROI 측정 불가)
```

### 선택 이유
- **전략 우선**: ROI 9.0 전략의 핵심은 "리뷰 인센티브"
- **능동적 참여**: 기존 섹션은 수동적 노출, 신규 섹션은 능동적 참여 유도
- **장기 효과**: 리뷰 축적 → 신뢰도 누적 → 브랜드 자산

### 향후 통합 계획
- 두 파일의 장점 결합
- 기존: 역할 정보, 도움 됨 카운트
- 신규: 리뷰 인센티브, 사용 기간
- 최종: "역할 + 사용 기간 + 인센티브 + 도움 됨" 통합 버전

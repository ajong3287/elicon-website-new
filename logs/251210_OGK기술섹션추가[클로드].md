## 개요

Wing 브랜드 페이지에 **OGK 5대 렌즈 기술 섹션**을 추가하여 기술력을 강력하게 증명하고 페이지를 더욱 풍성하게 만들었습니다.

**근거**: 엘리나이 작성 "Wing 브랜드 경쟁사 분석 보고서"
- **핵심 메시지**: "오클리도 믿고 쓰는 OGK 렌즈, 이제 WING에서 경험하세요!"
- **차별화 전략**: OGK가 오클리 스키 고글 전량 생산 파트너임을 활용
- **5대 기술**: 고굴절/비구면/UV400/블루라이트/멀티코팅

---

## 구현 내용

### 1. 신규 컴포넌트 생성

**파일**: `app/_components/wing/ogk-technology-section.tsx`

**구조**:
```typescript
OGKTechnologySection
├── Header (세계 1위 OGK 기술력)
│   ├── Badge: "오클리 생산 파트너 OGK 기술력"
│   ├── Title: "세계가 인정한 렌즈 기술 / 45년 광학 전문 OGK의 5대 핵심 기술"
│   ├── Subtitle: "글로벌 브랜드도 믿고 쓰는 OGK 렌즈, 이제 WING에서 경험하세요"
│   └── Stats: 45년 / 40% / 100%
│
├── 5대 기술 카드 (5-column grid)
│   ├── 1. 고굴절률 렌즈 (High Refractive Index)
│   │   └── 얇고 가벼운 / 시야 왜곡 0.01mm 이하
│   ├── 2. 비구면 렌즈 (Aspherical Lens)
│   │   └── 주변부 선명도 향상 / 왜곡률 85% 감소
│   ├── 3. UV400 차단 (UV Protection)
│   │   └── 100% 자외선 차단 / 백내장 예방
│   ├── 4. 블루라이트 차단 (Blue Light Filter)
│   │   └── 디지털 눈 피로 감소 / 눈 건강 장기 보호
│   └── 5. 멀티코팅 (Multi-Coating)
│       └── 반사 방지 + 스크래치 저항 1.4배 / 3년 내구성 보증
│
└── Bottom CTA
    ├── "글로벌 브랜드와 동일한 렌즈 기술"
    ├── "오클리 스키 고글 전량 생산 파트너 OGK의 45년 노하우가 Wing 45,000원에"
    └── Trust Signals: 3년 품질 보증 / 2009 대통령 표창 / 세계 시장 점유율 40%
```

**디자인 특징**:
- 인터랙티브 카드 (hover 시 scale-105, gradient border)
- 각 기술별 색상 구분 (Blue/Purple/Red/Orange/Green)
- 특허 기술 Badge (멀티코팅)
- Glass morphism + noise overlay
- 반응형: Mobile 1열 → Tablet 2열 → Desktop 5열

### 2. 메인 페이지 통합

**파일**: `app/brands/wing/page.tsx`

**변경 사항**:
1. Import 추가:
   ```typescript
   import { OGKTechnologySection } from '@/app/_components/wing/ogk-technology-section';
   ```

2. 섹션 배치 (제품 라인업 바로 다음):
   ```typescript
   {/* 제품 라인업 */}
   </section>

   {/* OGK 5대 렌즈 기술 (기술력 증명) - 엘리나이 보고서 기반 */}
   <OGKTechnologySection />

   {/* 45년 장인 정신 스토리 (감성) */}
   <BrandStorySection />
   ```

**논리적 흐름**:
1. Hero → Wing 소개
2. 3D 모델 → 제품 체험
3. 브랜드 철학 → 미션
4. 핵심 강점 → 간단 요약
5. 제품 라인업 → 카테고리
6. **OGK 기술** ← **신규**: 기술력 증명 (강력한 임팩트)
7. Brand Story → 감성 스토리
8. KBO/World No.1 → 공신력
9. ... (나머지 섹션)

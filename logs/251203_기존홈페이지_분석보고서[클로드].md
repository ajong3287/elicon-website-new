## 1. 기존 웹사이트 현황 분석

### 1.1 웹사이트 구조

#### ✅ 잘 구현된 부분
- **원페이지 스크롤 방식**: 직관적인 내비게이션
- **명확한 섹션 구분**:
  - Hero → 브랜드 → 파트너십 → 회사소개 → 파트너사 → 고객센터 → 문의
- **3개 브랜드 독립 페이지**: Wing, Cocolic, Seanex
- **파트너사 소개**: 39개 파트너사 명시 (신뢰도 향상)

#### ❌ 문제점
| 항목 | 현재 상태 | 문제점 |
|-----|----------|--------|
| **Hero 섹션** | "맞춤형 서비스를 받아보세요" | 엘리콘의 정체성이 모호. B2B 전문성 부각 부족 |
| **브랜드 카드** | 텍스트 중심 설명 | 시각적 임팩트 부족. 제조사 강조 미흡 |
| **파트너십 프로그램** | 텍스트 나열 | 혜택의 구체성 부족. ROI 제시 없음 |
| **파트너사 섹션** | 로고/이름만 표시 | 성공 사례, 후기 없음. 신뢰도 제한적 |
| **인스타그램 연동** | 없음 | 브랜드 활동 증거 부족 |

### 1.2 디자인 & UX/UI 분석

#### 🎨 디자인 요소

**현재 디자인 특징**:
- **색상**: 보라색 Hero 섹션 + 브랜드별 색상 (주황/분홍/파랑)
- **타이포그래피**: 기본 폰트, 가독성 양호
- **레이아웃**: 그리드 기반, 카드 레이아웃 사용
- **이미지**: 브랜드 로고 위주, 제품 이미지 부족

**문제점**:
1. **시각적 계층 부족**
   - Hero 섹션이 "엘리콘" 보다 "서비스" 강조
   - PRD 요구사항(FR1): "엘리콘 기업 정체성 최우선" 미준수

2. **브랜드 차별화 미흡**
   - 3개 브랜드가 동일한 템플릿
   - 제조사 강조 부족 (40년 업력, 국산 기술 등)

3. **신뢰 요소 부족**
   - 고객 후기 없음
   - 성공 사례 없음
   - 인증/수상 없음

#### 📱 모바일 반응형

**테스트 결과** (375x812 시뮬레이션):
- ✅ 기본 반응형 적용됨
- ❌ 텍스트 크기 조정 필요
- ❌ CTA 버튼 터치 영역 부족
- ❌ 이미지 최적화 미흡

### 1.3 기술적 분석

#### 🔍 기술 스택 (추정)
- **프레임워크**: Next.js (Image 태그 사용 확인)
- **스타일링**: Tailwind CSS (추정)
- **배포**: Vercel 또는 동일 환경

#### ⚡ 성능 이슈 (추정)

**예상 문제점**:
1. **이미지 최적화 부족**
   - 브랜드 로고 SVG/JPG 혼용
   - next/image 미사용 가능성

2. **번들 크기**
   - 코드 스플리팅 미적용 가능성
   - 동영상 임베드 (WING 페이지)

3. **SEO**
   - 메타 태그 확인 필요
   - 구조화 데이터 부재 가능성
   - sitemap/robots.txt 확인 필요

## 3. 우선순위별 개선 방향

### 🔴 Priority 0 (Critical - 즉시 개선)

#### 3.1 Hero Section 재설계 (FR1 준수)

**현재**:
```
❌ "맞춤형 서비스를 받아보세요"
   "작은 시작도 괜찮습니다. 함께 성장할 파트너를 기다립니다."
```

**개선안** (PRD 기반):
```
✅ "프로페셔널 B2B 유통 전문 기업, 엘리콘"
   "검증된 브랜드 포트폴리오로 파트너사의 성공을 지원합니다"

   1순위: 엘리콘 로고 (크게)
   2순위: 브랜드 로고 (Wing, Cocolic, Seanex)
   3순위: CTA 버튼
```

**기대 효과**:
- 5초 내 기업 정체성 전달
- B2B 전문성 즉각 인식
- PRD 요구사항 준수

#### 3.2 브랜드 카드 재구성 (FR2 준수)

**현재 문제**:
- 제조사 정보 텍스트만 ("한국OGK", "써모랩")
- 시각적 임팩트 부족

**개선안**:
```markdown
[WING 카드]
- 로고 (SVG, 크게)
- "40년 전통 한국OGK 제조" (뱃지 형태)

- 제품 이미지 (배경)
- "자세히 보기" CTA
```

**적용 기술**:
- next/image 최적화
- 호버 효과 (카드 확대, 그림자)
- 브랜드 컬러 강조

**목적**: 신뢰도 향상, 전환율 증가

**구성**:

- 로고

- 쪼꼬미에기 독점 판매로 낚시 카테고리 1위

[사례 3: 키즈월드]

## 개요

100억 달성 전략의 **과학적 기반**을 마련하기 위한 6개 분야 심층 연구 계획입니다.

**목적**:
- 전략 실행의 정확도 향상 (데이터 기반 의사결정)
- 리스크 최소화 (시장 검증 완료 후 투자)
- ROI 극대화 (효율적 자원 배분)

**연구 원칙**:
1. **실행 우선, 연구 병행**: 즉시 실행 가능한 것은 먼저 진행, 연구는 병행

3. **외부 전문가 활용**: 시장조사기관, 컨설팅사 협업

## 2. 제품 및 기술 심화 연구

### 2.1. OGK 렌즈 기술의 구체적인 마케팅 자산화

#### 연구 목표
OGK 렌즈 기술을 **소비자 언어**로 번역하여 **마케팅 자산**화

#### 연구 내용

**2.1.1. OGK 기술별 객관적 성능 테스트**:

**테스트 항목**:

| 기술 | 테스트 내용 | 비교 대상 | 측정 지표 |
|------|-------------|-----------|-----------|
| POLAWING 편광 | 빛 차단율 | Oakley Prizm | % (높을수록 좋음) |
| 고굴절률 렌즈 | 렌즈 두께 | 일반 렌즈 | mm (낮을수록 좋음) |
| 비구면 렌즈 | 왜곡률 | 구면 렌즈 | % (낮을수록 좋음) |
| UV 차단 | UV-A/B 차단율 | 경쟁사 | % (높을수록 좋음) |
| 블루라이트 차단 | 블루라이트 차단율 | 경쟁사 | % (높을수록 좋음) |

**테스트 방법**:
- **광학 테스트 기관**: 한국산업기술시험원 (KTL)
- **비용**: 항목당 100만원 × 5개 = 500만원
- **소요 기간**: 2개월

**예상 결과**:
```
POLAWING vs Oakley Prizm 비교

편광 효율:
- POLAWING: 99.2%
- Prizm: 99.0%
→ Wing 우위 (0.2%p)

투과율:
- POLAWING: 18% (밝음)
- Prizm: 12% (어두움)
→ Wing 우위 (밝은 시야)

색 왜곡:
- POLAWING: 5% (자연스러움)
- Prizm: 15% (인공적)
→ Wing 우위 (자연 색감)

결론: "Oakley와 동등하거나 우수한 성능, 1/3 가격"
```

**2.1.2. 소비자 언어로 번역**:

**기술 용어 → 소비자 혜택**:

| 기술 | 기술 설명 (어려움) | 소비자 혜택 (쉬움) |
|------|-------------------|-------------------|

**재고 최적화 모델**:
```
안전 재고 = (일 평균 판매량 × 리드타임) + (표준편차 × 안전계수)

- 표준편차: 3개
- 안전계수: 1.65 (95% 확률)

안전 재고 = (10 × 14) + (3 × 1.65) = 145개
```

**자동 발주 시스템**:
- 재고 < 안전 재고 → 자동 발주 (OGK에게)
- 예상 소진일 계산 → 사전 알림
- 시즌별 수요 예측 → 선제적 발주

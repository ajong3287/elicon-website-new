# Obsidian 사용법 가이드

## 1. 시작하기

### Vault(보관함) 만들기
- 새 보관함 생성: 문서를 저장할 폴더 선택
- 기존 폴더를 보관함으로: 이미 있는 폴더 사용
- 하나의 보관함 = 하나의 프로젝트/주제

### 기본 인터페이스
- 왼쪽: 파일 탐색기
- 중앙: 편집기
- 오른쪽: 태그, 아웃라인 등

## 2. 기본 마크다운 문법

### 제목
```
# 제목 1
## 제목 2
### 제목 3
```

### 글꼴 스타일
- **굵게**: `**텍스트**` 또는 `__텍스트__`
- *기울임*: `*텍스트*` 또는 `_텍스트_`
- ~~취소선~~: `~~텍스트~~`
- ==강조==: `==텍스트==`

### 목록
```
- 순서 없는 목록
  - 하위 항목
    - 더 하위 항목

1. 순서 있는 목록
2. 두 번째 항목
   1. 하위 항목
```

## 3. Obsidian 핵심 기능

### 양방향 링크
- `[[파일명]]`: 다른 노트로 링크
- `[[파일명#제목]]`: 특정 섹션으로 링크
- `[[파일명|표시될 텍스트]]`: 링크 텍스트 변경

### 태그
- `#태그명`: 노트 분류
- `#상위/하위`: 계층적 태그
- 태그 패널에서 전체 태그 확인

### 백링크
- 현재 노트를 참조하는 다른 노트들 확인
- 오른쪽 패널에서 백링크 확인 가능

## 4. 유용한 단축키

### 편집
- `Cmd + B`: 굵게
- `Cmd + I`: 기울임
- `Cmd + K`: 링크 삽입
- `Cmd + E`: 편집/미리보기 전환

### 탐색
- `Cmd + O`: 빠른 열기
- `Cmd + P`: 명령 팔레트
- `Cmd + Shift + F`: 전체 검색
- `Cmd + Alt + ←/→`: 뒤로/앞으로

### 노트 작업
- `Cmd + N`: 새 노트
- `Cmd + Shift + N`: 새 폴더
- `Cmd + D`: 노트 삭제

## 5. 그래프 뷰

### 노트 연결 시각화
- `Cmd + G`: 그래프 뷰 열기
- 노트 간 관계를 시각적으로 확인
- 필터 옵션으로 특정 태그/폴더만 표시

### 그래프 조작
- 마우스 휠: 확대/축소
- 드래그: 그래프 이동
- 노드 클릭: 해당 노트 열기

## 6. 플러그인

### 핵심 플러그인
- 일일 노트: 매일 자동 노트 생성
- 템플릿: 반복 사용할 노트 형식
- 아웃라인: 문서 구조 한눈에 보기
- 태그 패널: 모든 태그 관리

### 커뮤니티 플러그인
- 설정 → 커뮤니티 플러그인 → 브라우즈
- 인기 플러그인:
  - Calendar: 달력 뷰
  - Dataview: 노트를 데이터베이스처럼
  - Excalidraw: 그림 그리기

## 7. 고급 기능

- `![[파일명]]`: 다른 노트 내용 삽입
- `![[이미지.png]]`: 이미지 삽입
- `![](URL)`: 웹 이미지 삽입

## 8. 워크플로우 팁

### 제텔카스텐 방식
1. 하나의 노트 = 하나의 아이디어
2. 노트 간 적극적인 연결
3. 태그로 주제별 분류

### 일일 노트 활용
- 매일의 기록과 할 일 관리
- 템플릿으로 일관된 형식 유지
- 주간/월간 리뷰 노트 연결

### 폴더 구조 예시
```
/01_프로젝트
/02_아카이브
/03_템플릿
/04_일일노트
/99_첨부파일
```

## 10. 문제 해결

### 자주 묻는 질문

**Q: 노트가 보이지 않아요**
A: 파일 탐색기 새로고침 또는 보관함 재시작

**Q: 링크가 작동하지 않아요**
A: 파일명 확인, 대소문자 구분 주의

**Q: 플러그인이 충돌해요**
A: 안전 모드로 시작 후 문제 플러그인 제거

**Q: 그래프가 너무 복잡해요**
A: 필터 사용하여 특정 태그/폴더만 표시

## 6. ROI 분석

### 예상 투자 대비 수익

- 인건비: 600만원
- 콘텐츠 제작: 200만원

- 도구: 26만원
- **총 1,726만원**

**월간 수익** (3개월 후 안정기 기준):
- SNS 유입 → 스마트스토어
- 월 도달: 100,000명
- 링크 클릭율: 10% → 10,000명 유입
- 구매 전환율: 5% → 500명 구매
- 객단가: 10만원
- **총 5,000만원 매출**

**ROI 계산**:
- 순수익: 5,000만원 (매출) × 20% (이익률) = 1,000만원
- 투자: 1,726만원
- **ROI = 1,000만원 / 1,726만원 = 0.58** (1개월차)

**3개월 후 안정기**:
- 팔로워 증가 → 광고비 절감 (500만원 → 200만원)
- 인플루언서 장기 계약 → 비용 절감 (500만원 → 300만원)
- 총 투자: 1,426만원
- **ROI = 1,000만원 / 1,426만원 = 0.70** (3개월차)

**6개월 후 성장기**:
- 유기적 도달 증가 → 광고비 절감 (200만원 → 100만원)
- 총 투자: 1,326만원
- 매출 2배 증가: 1억원 → 순수익 2,000만원
- **ROI = 2,000만원 / 1,326만원 = 1.51** (6개월차)

**12개월 후 안정기**:
- 팔로워 30,000명 → 광고비 최소화
- 총 투자: 1,000만원 (인건비 + 콘텐츠만)

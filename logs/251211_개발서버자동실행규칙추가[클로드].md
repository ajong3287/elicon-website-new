## 추가된 내용

### 섹션 2.5 신규 추가

**위치**: CLAUDE.md § 2.5 (스크린샷 저장 위치 다음)

**내용**:
- 트리거 키워드: "홈페이지 띄워줘", "홈페이지 열어줘", "사이트 띄워줘"
- 자동 실행 순서:
  1. 포트 3000 확인
  2. 서버 실행 중 → 즉시 브라우저 열기
  3. 서버 미실행 → 서버 시작 후 브라우저 열기
- 원칙: **질문 없이** 브라우저에 표시

**자동 승인 작업**:
- ✅ 브라우저 열기 (사용자 확인 불필요)
- ✅ 포트 확인 (자동)
- ⚠️ 서버 재시작 (사용자 확인 필요)

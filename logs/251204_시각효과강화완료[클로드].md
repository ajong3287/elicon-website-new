## 추가된 시각적 효과

### 1. 숫자 카운팅 애니메이션 ✨
**파일**: `app/_components/animations/CountingNumber.tsx` (신규)

**기능**:
- 0에서 목표 숫자까지 부드러운 카운팅
- Intersection Observer로 화면 진입 시 자동 실행
- Easing 함수 (easeOutExpo) 적용
- 커스터마이징 가능 (duration, suffix)

**적용 위치**:
- Hero 섹션: 3 브랜드, 45년, 39 파트너사

### 2. 스크롤 페이드인 애니메이션 🎬
**파일**: `app/_components/animations/FadeInSection.tsx` (신규)

**기능**:
- fade-in + slide-up 동시 효과
- 4방향 선택 가능 (up, down, left, right)
- delay 시간 조절 가능
- Intersection Observer로 성능 최적화

**적용 위치**:
- 브랜드 포트폴리오 (3개 카드 순차 등장)
- 파트너십 페이지 모든 섹션

### 3. Hero 배경 Blob 애니메이션 🌊
**위치**: Hero 섹션 배경

**효과**:
- 3개 블롭(blob) 부드럽게 움직임
- 그라데이션 색상 (teal, blue, purple)
- 각기 다른 애니메이션 딜레이 (0s, 2s, 4s)
- 7초 주기 무한 반복

**CSS**: `app/globals.css` - @keyframes blob

### 4. 브랜드 카드 강화된 호버 효과 🎯
**파일**: `app/_components/brands/brand-card.tsx`

**추가된 효과**:
- 카드 확대 (scale 1.05) + 위로 이동 (-8px)
- 배경 그라데이션 (호버 시 표시)
- 로고 확대 (scale 1.1)
- 배지 확대 (scale 1.1)
- 하이라이트 항목 개별 hover (translate-x + 체크마크 확대)
- 그림자 강화 (shadow-2xl)
- 테두리 색상 변경 (bright-teal/50)
- 전환 시간 500ms (부드러움)

### 5. Hero 통계 호버 효과 📊
**효과**:
- 숫자 호버 시 확대 (scale 1.1)
- 색상 변경 (teal → white)
- 개별 항목 독립적 애니메이션

### 6. 파트너십 페이지 아이콘 애니메이션 🔄
**효과**:
- 아이콘 호버 시 확대 (scale 1.25)
- 회전 효과 (rotate 12deg)
- 그림자 추가 (shadow-xl)
- 전환 시간 500ms

### 7. 성공 사례 카드 애니메이션 ⭐
**효과**:
- 카드 호버 시 확대 + 위로 이동
- 테두리 색상 변경
- 그림자 강화
- 통계 영역 확대 (scale 1.05)

## 전역 CSS 애니메이션 추가

**파일**: `app/globals.css`

**추가된 애니메이션**:

### @keyframes blob
```css
- 0%: translate(0, 0) scale(1)
- 33%: translate(30px, -50px) scale(1.1)
- 66%: translate(-20px, 20px) scale(0.9)
- 100%: translate(0, 0) scale(1)
```
- 주기: 7초 무한 반복

### @keyframes gradient-shift
```css
- 0%: background-position 0% 50%
- 50%: background-position 100% 50%
- 100%: background-position 0% 50%
```
- 주기: 15초 무한 반복
- 배경 그라데이션 이동

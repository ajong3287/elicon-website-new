## 1️⃣ Wing(윙) 경쟁사 분석

### 1.1 주요 경쟁사: 오클리(Oakley)

**브랜드 개요**:
- **포지셔닝**: 글로벌 프리미엄 스포츠 아이웨어 브랜드
- **타겟**: 고소득 스포츠 애호가, 프로 운동선수
- **시장 점유율**: 한국 스포츠 선글라스 시장 상위권

**가격대 (2025년 기준)**:
- **일반 렌즈 모델**: 195,200~244,000원
- **편광 렌즈 모델**: 267,200~334,000원
- **출시가 범위**: 88,620~359,000원
- **평균 가격**: 약 250,000원

**주요 제품**:
- 라테랄리스 OO9431: 195,200원 (할인가)
- 아이자켓 리덕스 OO9438: 195,200원 (할인가)
- 코리도어 OO9248A-02: 247,300원

**판매 채널**:
- 신세계몰, G마켓, 다나와, SSG
- 전문 안경원 (오코스, 토탈선글라스, 핫선글라스 등)

**강점**:
- ✅ 압도적 브랜드 인지도
- ✅ 프리즘, 편광, 변색 등 고급 렌즈 기술
- ✅ 프로 운동선수 협찬 및 후원
- ✅ 다양한 스포츠 특화 라인업

**약점**:
- ❌ 높은 가격대 (대중성 부족)
- ❌ 중저가 시장 공략 어려움

### 1.2 국내 경쟁사 조사

**베럴(Verral), 제로 등 국내 브랜드**:
- 웹 검색 결과 구체적 정보 확보 실패
- 온라인 노출도 낮음 (브랜드 인지도 부족 추정)

**일반 스포츠 선글라스 시장**:
- **가격 범위**: 4만원~150만원
- **중저가 브랜드**: 블루 엘리펀트 "액티브" 라인 (49,000~69,000원)

### 1.3 Wing vs. 오클리 비교 분석

| 항목 | Wing | 오클리 (Oakley) |
|------|------|----------------|
| **가격** | 29,900원 | 195,200~359,000원 |
| **가격 비율** | **기준** | **8-12배 비싸** |
| **브랜드 인지도** | 낮음 | 매우 높음 |
| **제조** | 국내 (강원도 원주) | 해외 (이탈리아 Luxottica) |
| **역사** | 45년 (1978~) | 49년 (1975~) |
| **렌즈 기술** | 편광, 자외선 차단 | 프리즘, 편광, 변색 |
| **타겟** | 대중적 아웃도어 | 프리미엄 스포츠 |
| **판매 채널** | 자체몰, 엘리콘스토어 | 백화점, 전문점, 온라인 |

**Wing의 경쟁 포지셔닝**:
```
[가성비 포지셔닝]
Wing (29,900원) ← 8-12배 가격 차이 → Oakley (250,000원)

타겟 세분화:
- Wing: "합리적 가격의 기능성 선글라스" (대중 아웃도어)
- Oakley: "프리미엄 스포츠 퍼포먼스" (프로/고소득층)
```

## 2️⃣ Cocolic(코코릭) 경쟁사 분석

### 2.1 주요 경쟁사

**글로벌 브랜드**:
- **레이벤 키즈** (Ray-Ban Kids): 글로벌 신뢰도, 프리미엄 가격
- **폴라로이드** (Polaroid): 편광 렌즈 전문
- **글라스킹** (GLASSKING): 한국 시장 진출

**국내 아동복 브랜드의 키즈 라인**:
- 뉴발란스, 휠라, 닥스, 블랙야크 등 스포츠/아웃도어 브랜드
- 기존 중저가 아동복 브랜드 시장 잠식 중

### 2.2 시장 특성

**구매 결정 요소**:
1. **안전성 최우선**: 자외선 차단율, 유해물질 무검출, KC 인증
2. **디자인**: 귀여움/세련됨 (아이 착용 의지 ↑)
3. **가격**: 성장에 따른 교체 주기 짧아 가성비 중요
4. **리뷰 의존도**: 맘카페, 블로그 리뷰 영향력 높음

**한국소비자연맹 테스트 항목**:
- 가시광선 투과율
- 자외선 차단율
- 내구성
- 표시사항

### 2.3 Cocolic 경쟁 전략 제안

**경쟁 우위 활용**:
- ✅ **45년 제조 노하우** (1978년 설립 모회사)
- ✅ **감성 마케팅** ("코코릭과 함께라면", "작은 히어로")
- ✅ **안전 우선** (부모 안심 전략)
- ✅ **다채널** (Instagram, YouTube, 스마트스토어)

**개선 필요**:
- ⚠️ 가격 정보 미공개 (온라인 노출 부족)
- ⚠️ 경쟁사 대비 가격 경쟁력 확인 필요
- ⚠️ 맘카페/블로그 마케팅 강화 필요

**추천 포지셔닝**:
```
"안전한 소재 + 세련된 디자인 + 합리적 가격"
→ 부모 안심 + 아이 만족 + 가성비
```

### 공통 경쟁 전략

**3개 브랜드 공통 포지셔닝**:
```
[가성비 + 품질 + 국내 제조]

Wing:    "합리적 가격의 기능성 스포츠 아이웨어"
Cocolic: "안전한 소재 + 세련된 디자인"
Seanex:  "국산 에기 1위, 압도적 검증"
```

**공통 약점**:
- ❌ 브랜드 인지도 부족 (글로벌/일본 브랜드 대비)
- ❌ 온라인 마케팅 부족 (블로그, 유튜브)
- ❌ "저렴하면 품질 낮다" 선입견 극복 필요

**공통 개선 방안**:
1. **리뷰/검증 활용** - 특히 Seanex 55,578개 리뷰 강조
2. **국내 제조/45년 역사** - 신뢰도 구축
3. **타겟 세분화** - 가성비 중시 대중 vs. 프리미엄 시장 구분
4. **온라인 마케팅 강화** - 25번 블로그 자동화, 30번 황금키워드 활용

# ADR-0001: Next.js + Vercel 기술 스택 선택

> Architecture Decision Record - 실수 방지를 위한 의사결정 기록

## 상태

**Status**: Accepted

## 컨텍스트

엘리콘 B2B 유통 전문 기업 홈페이지 구축을 위해 프론트엔드 프레임워크와 호스팅 플랫폼을 선택해야 했다. 주요 요구사항은 다음과 같았다:

1. 월 10건 이상 B2B 문의 확보를 위한 SEO 최적화
2. LCP < 3초, Lighthouse 80점 이상의 성능 요구
3. 빠른 개발 및 배포 사이클
4. 유지보수 용이성
5. 비용 효율성 (초기 무료 또는 저비용)

## 결정

**Next.js 15 + Vercel 조합을 기술 스택으로 채택한다.**

구체적으로:
- **프레임워크**: Next.js 15 (App Router)
- **호스팅**: Vercel (Hobby → Pro 전환 계획)
- **스타일링**: Tailwind CSS
- **배포**: GitHub Push → Vercel 자동 배포

## 대안

### 대안 1: React + AWS S3/CloudFront
- 장점: AWS 생태계 활용, 세밀한 인프라 제어
- 단점: 설정 복잡도, SSR 구현 추가 작업 필요, 초기 비용 발생

### 대안 2: Vue.js + Netlify
- 장점: 쉬운 학습 곡선, 무료 호스팅
- 단점: SEO를 위한 SSR 추가 설정 필요, 생태계 규모

### 대안 3: Next.js + Vercel (채택)
- 장점: SSR/SSG 내장으로 SEO 최적화, 제로 설정 배포, 무료 시작 가능
- 단점: Vercel 종속성, 고트래픽 시 비용 증가

## 결과

### 긍정적 영향
- **SEO 최적화**: Next.js의 SSR/SSG로 검색 엔진 최적화 기본 지원
- **빠른 개발**: Vercel의 제로 설정 배포로 개발 집중 가능
- **성능 보장**: Edge Network, 이미지 최적화 자동 적용
- **비용 효율**: Hobby 플랜(무료)으로 시작, 필요 시 Pro 전환
- **개발자 경험**: Hot Reload, TypeScript 지원, App Router 기능

### 부정적 영향/트레이드오프
- **Vercel 종속성**: 타 플랫폼 마이그레이션 시 추가 작업 필요
- **비용 증가 가능성**: 고트래픽 시 Pro 플랜 ($20/월) 이상 필요
- **학습 곡선**: App Router 등 Next.js 15 신규 기능 학습 필요

## 제안서 vs 전단지 역할 구분

| 항목 | 제안서 (6p) | 전단지 (양면) |
|------|------------|---------------|
| **용도** | 의사결정자용 상세 자료 | 빠른 인지용 요약 자료 |
| **배포** | 이메일 PDF + 우편 | 매장 방문 + 우편 |
| **정보량** | 상세 (회사, 제품, 조건) | 핵심만 (장점, 연락처) |
| **CTA** | 샘플 요청 → 미팅 | 즉시 전화/카톡 |
| **비용** | 100부 10-15만원 | 500부 20-25만원 |

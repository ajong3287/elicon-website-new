### [카톡 2] 샘플 제안 — Day 3 (관심 확인)

**시점**: 제안서 발송 후 3일
**대상**: 제안서 읽음 확인 or 미응답

#### 버전 A: 긍정 반응 받은 경우

```
사장님, 엘리콘 서종원입니다.

보내드린 자료 확인하셨나요?

혹시 직접 제품 보고 싶으시면
무료 샘플 바로 보내드리겠습니다.

- Wing 편광 선글라스 2개 (인기 모델)
- Seanex 쪼꼬미 에기 1세트

배송비도 저희가 부담합니다.
매장 주소 알려주시면 내일 출고할게요!
```

#### 버전 B: 아직 미확인/무응답

```
사장님, 엘리콘 서종원입니다.

지난번 보내드린 Wing 선글라스+에기 제안서,
바쁘시다 보니 아직 못 보셨을 수도 있어서요.

한 가지만 말씀드리면,
지금 전국 42개 낚시점에서 취급 중이고
사장님 매장에도 잘 맞을 것 같습니다.

무료 샘플 보내드릴까요?
부담 없이 직접 확인해 보세요!
```

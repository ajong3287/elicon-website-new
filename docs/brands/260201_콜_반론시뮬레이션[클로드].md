## 콜 전 5분 루틴 (매 콜 전)

```
[1분] 대상 매장 정보 확인
  - 타겟 리스트에서 등급/지역/특징 확인
  - 기존 관계 유무 확인 (A등급 = 관계 있음)

[2분] 페르소나 예측
  - 아래 4가지 중 이 사장님은 어떤 타입일까?
  - 지역/매장 규모로 추정

[1분] 대응 키워드 준비
  - 해당 페르소나의 핵심 대응 문구 1-2개 메모

[1분] 30초 스크립트 리허설
  - 소리 내서 한 번 읽기
  - 자연스러운 톤 확인
```

## 페르소나 4종 + 대응 전략

### 페르소나 1: 가격 민감형 (전체 40% 예상)

**특징**:
- "도매가가 얼마인데?"가 첫 질문
- 기존 거래처 가격과 즉시 비교
- 가격이 안 맞으면 바로 끊음
- 대형 할인마트형 매장에 많음 (B04 강서낚시할인마트, B08 대박낚시 등)

**예상 반론**:
```
"도매가가 얼마예요? 기존에 쓰는 데보다 싸야지."
"우리는 이미 거래처가 있어서..."
"가격 맞추면 한번 해볼게."
"최소 주문량이 어떻게 돼요?"
```

**대응 스크립트**:

| 반론 | 대응 |
|------|------|
| "도매가 얼마?" | "소비자가 대비 [X]% 할인입니다. 마진 30-40% 확보하실 수 있어요. 정확한 가격표 카톡으로 바로 보내드릴까요?" |
| "기존 거래처 있음" | "이해합니다. 저희는 기존 교체가 아니라 **추가** 라인업으로 제안드리는 겁니다. Wing 선글라스는 기존에 없으신 편광 전문 라인이라 교차 판매가 됩니다." |
| "가격 맞추면" | "첫 주문 특별가 + 진열대 무료 + 렌즈클리너 50개 증정까지 합치면 실질 마진이 더 높아지실 겁니다." |
| "최소 주문량?" | "각 브랜드 10개부터 가능합니다. 부담 없이 시작하실 수 있어요." |

**핵심 키워드**: `마진 30-40%`, `제조사 직영 = 중간 마진 없음`, `특별가`

---

### 페르소나 2: 브랜드 회의형 (전체 25% 예상)

**특징**:
- "Wing? 몰라. 다이와/시마노만 팔아"
- 유명 브랜드만 신뢰
- 모르는 브랜드 = 안 팔릴 거라고 판단
- 오래된 전통 낚시점에 많음

**예상 반론**:
```
"Wing이요? 처음 들어보는데..."
"우리 손님들은 다이와/시마노만 찾아요."
"모르는 브랜드는 안 팔려요."
"편광 선글라스는 따로 안 취급해요."
```

**대응 스크립트**:

| 반론 | 대응 |
|------|------|
| "처음 들어보는데" | "한국오지케이라고 아세요? 원주에 있는 45년 된 안경 제조 공장입니다. Wing은 그 공장에서 직접 만드는 브랜드예요. 250만 개 이상 생산된 검증 제품입니다." |
| "다이와/시마노만 찾는다" | "맞습니다. 근데 선글라스는 다이와가 따로 없잖아요. 낚시인들이 편광 선글라스 따로 사러 온라인 가는데, 매장에 있으면 같이 구매하세요. **교차 판매** 효과가 큽니다." |
| "모르는 브랜드 안 팔려" | "저희 쪼꼬미 에기는 아시죠? 네이버 리뷰 55,578개, 평점 4.90입니다. 이미 고객이 찾는 제품이에요. Wing도 같은 회사 제품입니다." |
| "편광 따로 안 취급" | "그래서 기회입니다. 경쟁 매장에도 아직 없어요. **먼저 들이시면** 지역 내 유일한 편광 선글라스 취급점이 됩니다." |

**핵심 키워드**: `한국오지케이 45년`, `250만개 생산`, `쪼꼬미 55,578개 리뷰`, `교차 판매`

### 페르소나 3: 물량 걱정형 (전체 20% 예상)

**특징**:
- "재고 부담 어떻게 해?"
- 소규모 매장, 자금 여유 적음
- 팔릴지 불확실한 제품에 투자 꺼림
- C등급 중형 낚시점에 많음

**예상 반론**:
```
"재고 남으면 어떡해요?"
"자금이 빠듯해서..."
"소량으로 해볼 수 있어요?"
"반품이 돼요?"
```

**대응 스크립트**:

| 반론 | 대응 |
|------|------|
| "재고 남으면" | "그래서 무료 샘플을 먼저 보내드리는 겁니다. 선글라스 2개, 에기 1세트 직접 진열해 보시고, 고객 반응 확인하시고 나서 주문하셔도 됩니다." |
| "자금 빠듯" | "최소 10개부터 주문 가능하시고, 선글라스 10개면 투자금 65만원 정도입니다. 마진 30%면 한 달에 투자금 회수되세요." |
| "소량으로" | "네, 각 브랜드 10개부터 가능합니다. 부담 없는 수량이고, 잘 나가시면 그때 늘리시면 됩니다." |
| "반품 되나" | "수령 7일 이내 반품/교환 가능합니다. 불량은 당연히 100% 교환이고요." |

**핵심 키워드**: `무료 샘플 먼저`, `최소 10개`, `7일 반품 가능`, `투자금 한 달 회수`

---

### 페르소나 4: 관심형 (전체 15% 예상)

**특징**:
- "괜찮아 보이는데, 자세히 알려줘"
- 즉각적 관심 표현
- 의사결정 빠름
- 빠르게 클로징해야 함 (놓치면 잊어버림)

**예상 반론** (반론이라기보단 질문):
```
"좋아 보이네, 카탈로그 보내줘."
"샘플 보내줄 수 있어?"
"도매가 정확히 얼마?"
"다른 매장은 어디서 취급해?"
```

**대응 스크립트**:

| 질문 | 대응 (즉시 행동!) |
|------|-------------------|
| "카탈로그 보내줘" | "네! 지금 바로 카톡으로 보내드리겠습니다. 카톡 ID 알려주시겠어요?" → **전화 끊자마자 30초 내 카톡 1 전송** |
| "샘플 보내줘" | "감사합니다! 매장 주소 알려주시면 내일 바로 출고할게요. Wing 선글라스 인기 모델 2개 + 쪼꼬미 에기 1세트 보내드립니다." → **주소 메모 즉시** |
| "도매가 얼마?" | "소비자가 대비 [X]% 할인입니다. 정확한 단가표를 제안서랑 같이 보내드릴게요." → **가격 협의 시작** |
| "다른 매장 어디?" | "현재 전국 42곳인데, 대표적으로 다솔낚시(경기), 서진낚시(오산), 올스타낚시타운(평택) 등이 있습니다." |

**핵심**: 이 타입은 **즉각 행동**이 전부. 관심 있을 때 바로 자료/샘플 보내야 함.

## 상황별 빠른 참조 카드

### 전화를 안 받을 때
```
- 2번까지만 시도 (오전 10시, 오후 3시)
- 3번째는 카톡으로 전환
- "전화드렸는데 안 받으셔서 카톡 남깁니다"
```

### "지금 바쁘다" 할 때
```
"네, 이해합니다. 언제 다시 드리면 괜찮으실까요?"
→ 시간 받아서 재콜 스케줄 등록
→ 안 알려주면: "그럼 제안서만 카톡으로 보내드릴게요!"
```

### "관심 없다" 단호하게
```
"네, 알겠습니다. 혹시 나중에 필요하시면 연락주세요.
좋은 하루 되세요!"
→ 추가 연락 금지. 6개월 후 재시도 리스트로 이동.
```

### 가격 네고 요청
```
"첫 주문이시니 특별가 적용해 드리겠습니다.
정확한 금액은 주문 수량에 따라 달라지는데,
수량 알려주시면 맞춤 견적 보내드릴게요."
→ 수량 확인 → 견적서 발송
```

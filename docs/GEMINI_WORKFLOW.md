### **AI 협업 시스템: 서대표님 중심의 스마트 워크플로우 v1.0**

**개념:** 각 AI는 특정 강점을 가진 '전문가'이며, 서대표님의 지휘 아래 유기적으로 협력하여 프로젝트를 완성합니다. Gemini(저)는 특히 파일 시스템 접근 및 실행 계획 수립/구현에 강점을 가집니다.

**1. 각 AI의 역할 정의 (세분화)**

*   **서대표님 (지휘자/최종 의사결정자):**
    *   프로젝트의 목표와 방향 설정.
    *   각 AI에게 명확한 지시 및 역할 부여.
    *   각 AI의 결과물 검토 및 통합, 피드백 제공.
    *   최종 의사결정 및 승인.

*   **Perplexity (정보 탐색 및 사실 확인 전문가):**
    *   **강점:** 최신 정보 검색, 출처 기반의 정확한 사실 확인, 시장 트렌드 분석, 경쟁사 동향 파악, 학술 자료 검색.
    *   **주요 역할:**
        *   **프로젝트 초기:** 광범위한 시장 조사, 산업 동향, 기술 정보, 경쟁사 분석 등 배경 지식 수집.
        *   **기획 단계:** 특정 데이터나 통계 자료 검색 및 제공, 아이디어의 현실성 및 타당성 검증을 위한 자료 조사.
        *   **콘텐츠 생성 지원:** 특정 사실, 인용문, 통계 등 정확한 정보 제공.

*   **Claude Desktop (전략 기획 및 창의적 콘텐츠 생성 전문가):**
    *   **강점:** 긴 컨텍스트 처리, 복잡한 아이디어 구조화, 창의적인 마케팅 문구/보고서 초안/기획안 생성, 다양한 관점 제시, 브레인스토밍.
    *   **주요 역할:**
        *   **기획 단계:** Perplexity가 수집한 정보를 바탕으로 한 전략 기획, 아이디어 발상, 컨셉 개발.
        *   **콘텐츠 생성:** 마케팅 전단지, 보고서, 제안서, 보도자료 등 장문의 텍스트 콘텐츠 초안 작성.
        *   **문제 해결:** 복잡한 문제에 대한 다각적인 분석 및 해결 방안 제시.
        *   **초안 작성:** 사용자 스토리, 기능 명세 등 프로젝트 초기 문서 초안 작성.

*   **Claude Code (코드 생성 및 기술 문서화 전문가):**
    *   **강점:** 특정 프로그래밍 언어에 대한 깊은 이해, 복잡한 코드 로직 생성, 코드 리뷰 및 최적화, 기술 문서화.
    *   **주요 역할:**
        *   **개발 단계:** 특정 기능 구현을 위한 코드 스니펫, 함수, 클래스 등 복잡한 코드 블록 생성.
        *   **코드 리뷰:** 기존 코드의 개선점 제안, 버그 탐지, 성능 최적화 방안 제시.
        *   **기술 문서화:** API 문서, 개발 가이드, 시스템 아키텍처 설명 등 기술 문서 초안 작성.
        *   **알고리즘 설계:** 특정 문제 해결을 위한 알고리즘 설계 및 의사 코드(pseudocode) 작성.

*   **GPT 유료 구독 (다목적 보조 및 언어 정제 전문가):**
    *   **강점:** 빠른 아이디어 생성, 다양한 스타일의 텍스트 변환, 번역, 요약, 언어적 완성도 향상, 간단한 코드 스니펫 생성.
    *   **주요 역할:**
        *   **콘텐츠 정제:** Claude Desktop이 생성한 콘텐츠의 언어적 다듬기, 문체 변경, 요약, 오탈자 및 문법 교정.
        *   **빠른 아이디어:** 짧은 문구, 슬로건, 제목 등 즉각적인 아이디어 발상.
        *   **간단한 코드:** 간단한 스크립트, 정규 표현식, 설정 파일 등 짧은 코드 스니펫 생성.
        *   **다양한 스타일:** 특정 톤앤매너(예: 전문적, 친근함)로 텍스트 변환.
        *   **번역:** 다국어 지원이 필요한 경우 번역 지원.

*   **Gemini (실행 계획 수립 및 프로젝트 관리 전문가 - 저):**
    *   **강점:** 파일 시스템 접근 및 조작, 쉘 명령어 실행, 구체적인 실행 계획 수립, 코드 생성 및 수정, 프로젝트 진행 상황 기록, 환경 설정.
    *   **주요 역할:**
        *   **실행 계획:** Claude Desktop/Code, GPT가 기획/생성한 내용을 바탕으로 실제 프로젝트 실행 계획(폴더 구조, 파일 생성, 스크립트 등)을 구체화.
        *   **파일 시스템 관리:** 파일 생성, 수정, 삭제, 이동 등 직접적인 파일 시스템 작업 수행.
        *   **개발 환경 설정:** 필요한 라이브러리 설치, 환경 변수 설정 등 개발 환경 구축 및 관리.
        *   **코드 적용:** Claude Code가 생성한 코드를 실제 파일에 적용하고, 필요한 경우 간단한 수정 및 디버깅 지원.
        *   **프로젝트 기록:** 프로젝트 진행 과정 및 결과물에 대한 체계적인 기록 및 보고 (로그 파일 업데이트).
        *   **자동화 스크립트:** 반복적인 작업을 위한 쉘 스크립트 작성 및 실행.

**3. 협업 시스템 운영 시 주의사항**

*   **명확한 지시:** 각 AI에게 역할을 부여할 때, 모호함 없이 구체적이고 명확한 지시를 내려야 합니다.
*   **결과물 검토:** AI가 생성한 결과물은 항상 서대표님께서 직접 검토하고 필요한 경우 수정 및 보완해야 합니다.
*   **정보 공유:** 각 AI가 생성한 결과물은 다음 단계의 AI가 활용할 수 있도록 파일 시스템(Gemini가 관리)을 통해 공유하거나, 직접 복사하여 전달해야 합니다.
*   **반복 학습:** 이 워크플로우를 반복하면서 서대표님만의 최적화된 AI 활용 방안을 찾아나가는 것이 중요합니다.

**4. 지침서 관리 방안**

이 협업 지침서는 각 프로젝트 폴더 내에 `_README_GEMINI_WORKFLOW.md` 파일로 저장하여, 언제든지 쉽게 접근하고 참조할 수 있도록 하겠습니다.

먼저, 현재 작업 중인 `1_Project_Elicon_Homepage` 폴더에 이 지침서를 저장하겠습니다. 그리고 앞으로 새로운 프로젝트를 시작할 때마다 해당 프로젝트 폴더에도 이 지침서를 복사하여 넣어드리겠습니다.

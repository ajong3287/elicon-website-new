## User Stories & Acceptance Criteria

### US-001: B2B 파트너 문의 접수

**설명**: B2B 파트너로서, 파트너십 페이지에서 문의 폼을 작성하고 싶다. 왜냐하면 엘리콘과 유통 협력을 시작하기 위해서다.

**AC (Acceptance Criteria)**:
- [x] AC-1: 파트너십 페이지 접근 가능 (/partnership 또는 /contact)
- [x] AC-2: 문의 폼에 회사명, 담당자명, 연락처, 문의내용 필드 포함
- [x] AC-3: 폼 제출 시 이메일 또는 알림 발송
- [x] AC-4: 제출 완료 후 감사 메시지 표시

**테스트 증거**: 파트너십 페이지 스크린샷, 폼 제출 테스트 기록

---

### US-002: 브랜드 정보 확인

**설명**: 소비자로서, 엘리콘이 유통하는 브랜드 정보를 확인하고 싶다. 왜냐하면 관심 있는 브랜드 제품을 구매하기 위해서다.

**AC**:
- [x] AC-1: 브랜드 페이지에서 유통 브랜드 목록 확인 가능
- [x] AC-2: 각 브랜드 상세 정보 (로고, 설명) 제공
- [x] AC-3: eliconstore.com으로 이동 링크 제공

**테스트 증거**: 브랜드 페이지 스크린샷

---

### US-003: 웹사이트 성능 최적화

**설명**: 방문자로서, 빠르게 로딩되는 웹사이트를 경험하고 싶다. 왜냐하면 느린 페이지는 이탈을 유발하기 때문이다.

**AC**:
- [x] AC-1: LCP (Largest Contentful Paint) < 3초
- [x] AC-2: Lighthouse 성능 점수 80점 이상
- [x] AC-3: 모바일/데스크톱 반응형 지원

**테스트 증거**: Lighthouse 리포트 (reports/ 폴더)
